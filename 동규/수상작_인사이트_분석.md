## 1. 서사 구조 인사이트

### 1-1. 제목: 도발적 질문으로 시작한다

두 수상작 모두 **사회적 긴장감을 유발하는 도발적 제목**으로 서사를 이끌었다.

- 대상: "아이 대신 반려동물?" — 저출산이라는 사회 문제를 뒤집는 시각
- 최우수상: "노인은 어디서 노는가?" — '논다'는 표현으로 노인 이동을 경쾌하게 프레이밍

**인권 프로젝트 적용:**
- "교권과 학생 인권, 정말 제로섬인가?" 또는 "교사가 침해받는 학교, 아이들 인권은 안전한가?"처럼 직관적 긴장감을 제목에 담을 것
- 제목 슬라이드에 아이콘/일러스트 1개 + 부제목 구조 유지 (대상작 방식)

### 1-2. 슬라이드 서사 흐름

**대상작 구조 (11슬라이드):**
```
01 문제 배경 (통계/뉴스 인용)
02 문제 제기 (연구 질문 명확화)
03 데이터 선택과 분석 방법 (3단 구성: 궁금증 → 데이터 → 분석방법)
04 인과관계 검증 (시계열 그래프)
05 주요 발견 1 (비율 인포그래픽)
06 주요 발견 2 (집단 비교 통계)
07 주요 발견 3 (2차 분석)
08 예측 분석 (회귀)
09 토의 1 (인과 오류 / 반려동물 불평등 / 세대 과제)
10 토의 2 (한계 및 향후 과제)
11 참고문헌
```

**최우수상 구조 (9슬라이드):**
```
01 문제 제기
02 데이터 선택 및 분석방법 (표로 정리)
03 노인의 하루 (기술 통계)
04 지역사회시설 이용 실태 (레이더 차트)
05 지하철 무임승하차 지도 (공간 분석)
06 생활편의시설과 무임하차 주요역 (히트맵)
07 회귀분석
08 데이터 결과 정리 (번호 매긴 4가지 발견)
09 결론 및 제언
```

**인권 프로젝트 적용:**
- 슬라이드 수는 10~12개 적정
- 발견 슬라이드마다 슬라이드 상단에 "이 슬라이드의 한 문장 결론" 먼저 제시 (최우수상 방식)
- 결과 정리 슬라이드: 번호 매긴 4~5개 발견을 간결하게 나열 (최우수상 08번 슬라이드 방식)

## 2. 데이터 전략 인사이트

### 2-1. KOSSDA + 외부 데이터 결합이 수상의 핵심

두 작품 모두 **KOSSDA 단독에 머물지 않고** 공공 오픈데이터를 추가했다.

| 역할 | 대상작 | 최우수상 | 인권 프로젝트 적용 |
|------|--------|---------|----------------|
| 맥락 데이터 | 농림부 반려동물 실태조사 (추이) | 서울시 지하철 승하차 빅데이터 | 교육부 교원 현황 통계 / 아동권리위 연간 보고서 |
| 인구 구조 | 통계청 인구총조사 | 통계청 연령대별 행위자 시간 | KOSIS 교원 수, 학교 수, 학생 1인당 교원 수 |
| 지역 맥락 | 시/도별 출산율 (KOSIS) | 서울시 생활시설 분포 (오픈데이터) | 시도별 교원 1인당 학생 수 (E1), 아동학대 신고 건수 (E2) |
| 경제·사회 | 가구 소득/지출 (가계동향조사) | — | 지역별 교육 예산 (E3) |

### 2-2. 데이터 소개 슬라이드 포맷

최우수상이 사용한 **표 형태 데이터 소개**가 심사위원 신뢰도를 높인다.

```
| 데이터명 | 제공 출처 | 사용한 변수 | 데이터 시점 |
|---------|---------|-----------|-----------|
```
- 분석 데이터(D1~D4)와 보조 데이터(E1~E3) 명확히 구분해 표시

## 3. 분석 기법 인사이트

### 3-1. 대상작: 2×2 집단 비교 → 비모수 검정 → 회귀 순서

**핵심 설계: 반려동물 유무 × 자녀 유무 2×2 그룹**

인권 프로젝트에서 동일하게 적용할 수 있는 그룹 설계:

| 그룹 설계 | 인권 프로젝트 응용 |
|---------|----------------|
| 반려동물 유무 × 자녀 유무 | **교권침해 경험 유무 × 인권교육 이수 여부** |
| 반려동물 유무 × 추가 자녀 계획 | **침해 유형(언어/신체/정서) × 교급(초/중/고)** |
| 1인 가구 여부 | **경력 구간 (5년 미만/5~10/10년 이상)** |

**검정 방법 선택 기준 (대상작이 명시):**
- Likert 척도 → **Brunner-Munzel 검정** (분산 동질성 불가정, Mann-Whitney보다 robust)
- 연속형 정규 분포 → t-test
- 범주형 독립성 → Chi-square

**Density Plot + Violin Plot 조합:**
- 집단 간 분포 차이를 한 화면에 표현
- 음영 채운 Density + Violin + Box의 3-in-1 구성이 심사위원에게 설득력 높음
- 핵심 p-value는 그래프 위에 직접 표기

### 3-2. 최우수상: GIS 공간 분석 → 히트맵 → 회귀 순서

**지도 시각화 파이프라인:**
```
주소 데이터 → GIS 지오코딩 → 위경도 포인트 레이어
→ 히트맵 (밀도) → 행정구역 집계 → 단계구분도(Choropleth)
→ 독립변수로 회귀분석 투입
```

**인권 프로젝트 공간 분석 적용:**
- 시도별 교원 인권침해 경험률 → Choropleth Map
- 교원 1인당 학생 수(E1) × 침해율 산점도 + 회귀선
- 지역별 인권교육 시간 × 침해율 히트맵

**레이더 차트 3-in-1 구성 (최우수상 04번 슬라이드):**
- 동일 개념을 3가지 지표(이용 지표 / 빈도 지표 / 접근성 지표)로 나눠 3개 레이더 차트 병렬 배치
- 인권 프로젝트 적용: **인권 역량 5차원** (인지/정서/행동/교육환경/제도신뢰)을 현직교원 vs 예비교원으로 비교

### 3-3. 공통 회귀 전략

| 종속변수 | 독립변수 | 모델 | 수상작 참조 |
|---------|---------|------|-----------|
| 반려동물 키울 가능성 (0/1) | 사회활동, 거주형태, 취업여부, 가구구성 | 로지스틱 회귀 | 대상작 |
| 무임승하차 수 | 인근 500m 병원·약국·재래시장 수 | 선형 회귀 | 최우수상 |
| **인권침해 경험 여부 (0/1)** | **인권교육 이수, 경력, 교급, 지역** | **로지스틱 회귀** | 대상작 적용 |
| **인권 감수성 점수** | **인권교육 시간 수** | **선형 회귀 (OLS)** | 최우수상 적용 |

**상관행렬 활용 (최우수상 07번 슬라이드):**
- 다중공선성(VIF) 먼저 확인 → 5 이하 확인 후 회귀 투입
- 상관행렬 히트맵으로 변수 간 관계 먼저 시각화한 뒤 회귀 결과 제시

## 4. 시각화 패턴 인사이트

### 4-1. 인포그래픽 숫자 강조 (대상작 핵심 기법)

대상작 05번 슬라이드 방식:
```
자녀 있는 가구   자녀 없는 가구
반려동물 비율    반려동물 비율
  12.92%    <    16.64%
```
- **큰 폰트 숫자 + 부등호**로 집단 차이를 한눈에 전달
- 색상: 대조색 2가지만 사용 (단색 계열 + 강조색)
- 아이콘(가족/반려동물)으로 그룹 시각적 구분

**인권 프로젝트 적용:**
```
인권교육 이수 교원    인권교육 미이수 교원
침해 경험률          침해 경험률
  XX.X%       <      XX.X%
```

### 4-2. 슬라이드 레이아웃 원칙

| 원칙 | 대상작 | 최우수상 |

## 5. 토의·한계·제언 슬라이드 인사이트

### 5-1. 대상작 토의 구조 (3컬럼 병렬)

대상작은 토의를 2개 슬라이드로 구성, 각각 3개 컬럼:
- **토의 1 (09번)**: 인과관계 오류 경계 / 반려동물 불평등 / 세대적 과제
- **토의 2 (10번)**: 어느 특정집단 가능성 높은가 / 반려동물 불평등 / 연구의 한계와 기여

**인권 프로젝트 적용:**
- 토의 1: 횡단면 편향 (인과 추론 한계) / 자기보고 편향 / 지역변수 한계
- 토의 2: RQ2 시사점 (교권-학생인권 상생 or 제로섬) / RQ3 시사점 (교육 효과) / 정책 제언

### 5-2. 최우수상 결론 구조 (번호 매긴 4가지 발견)

```
1. 노인은 도시의 '활동적 주체'다. — 데이터 근거 1문장
2. 노인은 지역 사회시설을 제한적으로 이용한다. — 데이터 근거 1문장
3. 노인의 이동은 특정 생활편의시설 밀집 지역에 집중되어 있다. — 데이터 근거 1문장
4. 노인의 이동은 생활밀착형 시설의 분포와 통계적으로 유의미한 상관관계를 보였다. — 데이터 근거 1문장
```

**인권 프로젝트 적용 (결과 정리 슬라이드):**
```
1. RQ1: 교원 인권침해 경험률은 2021→2024 기간 [증가/감소]했다.
2. RQ2: 교권침해가 높은 집단은 학생 인권교육 만족도도 [낮은/비슷한] 경향을 보인다.
3. RQ3: 인권교육 이수 경험은 침해 대응 역량과 [유의미한/유의미하지 않은] 관계를 보인다.
4. 지역별: [특정 시도]에서 침해율이 통계적으로 유의미하게 높다.
```

## 6. RQ별 분석 설계 적용 체크리스트

### RQ1: 2021→2023→2024 추이 분석

- [ ] D3(2021) → D2(2023) → D1(2024) 종단 비교셋 구성 (연도 컬럼 추가 후 rbind)
- [ ] 변수 매핑 테이블 작성 (동일 개념 문항 크로스 연도 확인)
- [ ] **꺾은선 그래프**: 연도별 인권침해 경험률 추이 — 대상작 반려동물 등록수 추이 차트와 동일 방식
- [ ] 연도별 점 강조 + 수치 레이블 표기

### RQ2: 교권-학생 인권 상생/제로섬 분석

- [ ] **2×2 그룹 설계**: 교권침해 경험 유무 × 학생 인권교육 만족도 수준 (상/하위)
- [ ] **Brunner-Munzel 검정** (Likert 척도) — 대상작 직접 적용
- [ ] **Density + Violin Plot** 병렬 비교 — 대상작 06번 슬라이드 방식
- [ ] 집단 간 p-value 그래프 위 표기
- [ ] 인포그래픽: 교권침해 높은 학교 vs 낮은 학교의 학생 인권 만족도 수치 비교
